FAQ

What’s included in “automated updates”?

WordPress core, theme, and plugin updates are applied automatically within a controlled workflow. If an update introduces instability, we revert using backups/restore points. To keep this reliable, we enforce a plugin policy and avoid unsupported customization in the base hosting layer.

Do you support any plugin/theme?

Not universally. We support a standard set that behaves predictably in an automated environment. Unsupported or high-risk plugins may be declined, replaced, or moved to a custom scope to protect platform stability.

Is ecommerce possible without a lot of overhead?

Yes, if it is tightly scoped. We offer a premium ecommerce add-on with strict guardrails (Stripe-only, standard WooCommerce setup, limited plugin surface area). This reduces security and support risk and makes it profitable.

What happens if I want to leave?

You own your content. We can provide a full site export and database dump. If you have a custom domain, you can repoint DNS to another provider at any time.
